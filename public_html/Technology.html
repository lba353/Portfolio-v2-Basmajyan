Technology

These are the programs I used to create this website.

HTML (HyperText Markup Language)

HTML is a computer programming software that allows
you to make multiple things, such as your own website
or online games.

CSS (Cascading Style Sheet)

CSS is a style software that you can change the
background color, the color of the text and more
to make your website just the design you want.

Bootstrap

Bootstrap, which has to be downloaded, is a type of
software that gives the programmers a chance to use
many new tools, such as a drop down bar, a carousel,
and an accordion (collapsed).

Tiled

Tiled is a map editor that is easy to use and can be
downloaded. With Tiled, you can make your level the
way you want it to be with ease.
